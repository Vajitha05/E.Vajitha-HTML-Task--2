1.Understand the difference between block-level and inline elements. Practice examples with <div>, <p> (block) and <span>, <a> (inline).

Ans: Block level Elements:
     * Block level element always start on a new line.
     * Block level element take full width of the page.
     * Block-level example: <div>,<p>
      
     Inline Elements:
     *Inline element do not start on a new line
     *Inline elements take only required spaces
     *inline example:<span>,<a> 
Example:
    <!DOCTYPE html>
<html>
<head>
<title>Block and Inline Elements</title>
</head>
<body>
  <div>Welcome to HTML learning</div>
  <p>This is a paragraph example for block element.</p>
<span>This is a span inline element.</span>
<a href="https://www.mywebsite.com">Click here to visit website</a>
</body>
</html>
